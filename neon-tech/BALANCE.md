# Balance Reference

All tunable gameplay values live in `src/game/constants.js`.

## Match
- `match.duration` (seconds)
- `match.extractionOpen` (seconds)
- `match.extractionClose` (seconds)

## Player
- `player.speed`
- `player.dashSpeed`
- `player.dashDuration`
- `player.dashCooldown`
- `player.health`
- `player.reloadTime`

## Roles
- `hacker.visionDuration`
- `hacker.cooldown`
- `bruiser.shieldDuration`
- `bruiser.cooldown`
- `bruiser.damageReduction`

## Weapons
- `weapons.pistol.damage`
- `weapons.pistol.fireRate`
- `weapons.pistol.range`
- `weapons.pistol.spread`
- `weapons.rifle.damage`
- `weapons.rifle.fireRate`
- `weapons.rifle.range`
- `weapons.rifle.spread`
- `weapons.rifle.magazine`
- `weapons.rifle.reserve`

## Bots
- `bot.health`
- `bot.speed`
- `bot.sightRange`
- `bot.fireRate`
- `bot.damage`

## Loot
- `pickups.lootValue`

## Upgrades
- `upgrades.storage[]` (loot capacity per level)
- `upgrades.scanner[]` (minimap radius per level)
- `upgrades.workbench[]` (damage multiplier per level)
